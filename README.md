# shush 🤫

> "shh, don't leak my secrets."

A local-first secret manager for developers. Built for the VibeCoding era where AI agents can accidentally push your API keys.

## The Problem

- `.env` files with API keys get accidentally committed
- AI coding agents (`git add .` → `git push`) don't know what's sensitive
- Cloud-based secret managers require trusting a third party with your keys
- Existing local tools (KeePass, etc.) have poor developer UX

## The Solution

**shush** keeps your real secrets outside your project directory — forever.
Projects only ever contain *references*, not real values.

```bash
# Instead of this in your .env:
OPENAI_KEY=sk-abc123...   # ← dangerous

# shush gives you this:
OPENAI_KEY=@vault:openai  # ← safe to commit, safe to push
```

Even if an AI agent pushes your `.env`, there's nothing to leak.

## How It Works

```bash
# Add a secret to your local vault
shush add openai sk-abc123...

# Run any command with secrets injected as env vars
shush run npm start
shush run python main.py

# Scan for leaked secrets (used in git hooks)
shush scan

# Install git pre-commit hook + Claude Code hook automatically
shush install-hooks
```

## Sharing with Teammates

No server. No cloud. Uses public-key cryptography (compatible with GitHub SSH keys).

```bash
# Share a secret with a teammate (encrypted with their GitHub public key)
shush share openai --to bob

# Bob receives and imports it
shush receive <encrypted-blob>
```

The encrypted blob can be sent via Slack, email, GitHub Issues — it doesn't matter.
Only Bob can decrypt it.

## Architecture

- **Language:** Rust
- **Encryption:** [age](https://github.com/FiloSottile/age) (compatible with SSH keys)
- **Storage:** SQLite + SQLCipher (`~/.shush/vault.db`)
- **Key unlock:** macOS Keychain / Touch ID (Linux: master password + Argon2id)
- **Daemon:** Unix socket (`~/.shush/shush.sock`) for low-latency secret injection
- **Distribution:** `cargo install` / Homebrew tap

## Vault Location

```
~/.shush/
├── vault.db       # Encrypted secrets (never inside any git repo)
├── shush.sock     # Unix socket (daemon)
└── config.toml    # Non-sensitive config
```

## AI Tool Integration

shush installs hooks for AI coding tools to prevent accidental secret leaks:

```jsonc
// .claude/settings.json (auto-generated by `shush install-hooks`)
{
  "hooks": {
    "PreToolUse": [{
      "matcher": "Bash",
      "hooks": [{
        "type": "command",
        "command": "shush scan --block-on-secret"
      }]
    }]
  }
}
```
